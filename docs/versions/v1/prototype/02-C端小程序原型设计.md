# 摊点通 V1 C 端小程序原型设计

## 1. 信息架构

底部导航固定为“首页、订单、我的”。商品订单进入“订单”，服务预约进入“我的预约”。每个商户独立保存购物车。

## 3. C-01 商户首页

### 页面目标

让顾客快速理解商户状态并完成购买或预约。商户可选择商品优先或服务优先模板。

### 页面结构

```text
┌─────────────────────────┐
│ 春风小铺      营业中     │
│ 上海路 18 号 · 09:00-20:00│
├─────────────────────────┤
│      轮播图 16:9         │
├─────────────────────────┤
│ 商品分类：推荐 咖啡 轻食   │
├───────────┬─────────────┤
│ 桂花拿铁   │ 手作三明治    │
│ ¥18       │ ¥22          │
│ [选规格]   │ [加入购物车]  │
├─────────────────────────┤
│ 热门服务：咖啡体验课       │
│ 最近可约：今天 15:00      │
│             [立即预约]    │
├─────────────────────────┤
│ 首页          订单    我的 │
└─────────────────────────┘
```

服务优先模板交换商品区与服务区顺序，并扩大服务卡片。暂停营业时顶部显示提示条，禁止新订单和预约，但允许查看历史记录。

### 状态

- 未配置轮播图：显示商户头图。
- 商品为空：展示服务内容和“暂无在售商品”。
- 商户待启用预览：显示固定“预览模式”，禁止交易。
- 商户停用：显示暂停服务页，不展示交易按钮。

## 4. C-04 规格与加料选择

使用底部弹层展示商品图片、基础价格、规格分组、加料分组、数量和合计金额。必选组必须完成选择才能加入购物车；达到上限时禁用其他选项并说明原因。

```text
桂花拿铁                         关闭
杯型（必选）  [中杯] [大杯 +¥3]
温度（必选）  [热] [少冰] [正常冰]
加料（最多选 2 项）
[桂花冻 +¥2] [燕麦奶 +¥3] [奶油 +¥2]
数量          [-] 1 [+]
合计 ¥23                [加入购物车]
```

## 5. C-05 与 C-06 购物车和确认订单

购物车按商品展示规格、加料、单价、数量和小计。商品下架、售罄或配置变化时标记“已失效”，顾客必须删除或重新选择。

确认订单包含：

- 商户与门店。
- 商品和价格快照预览。
- 联系电话，支持微信授权或手动填写。
- 预计取餐时间；商户未启用时只显示“支付后请尽快到店领取”。
- 商品金额、加料金额和实付金额。
- 退款与取餐说明。
- “提交订单并支付”按钮。

提交中禁止重复点击。价格或库存变化时返回购物车并明确展示变化项。

## 6. C-08 取餐凭证

```text
┌─────────────────────────┐
│ 支付成功 · 待取餐        │
│                         │
│        A023             │
│      请留意商户叫号       │
│ 预计取餐：今天 15:30     │
│                         │
│       核销二维码          │
│ 请勿提前提供给他人        │
├─────────────────────────┤
│ 桂花拿铁×1  三明治×1      │
│ 春风小铺 · 上海路 18 号   │
│ [查看订单详情] [申请退款] │
└─────────────────────────┘
```

已核销后隐藏有效二维码，显示“已于今天 15:42 完成核销”。退款中、已退款和二维码加载失败均有独立状态。

## 7. C-09 与 C-10 商品订单

订单列表页签为“全部、待支付、待取餐、已完成、退款”。卡片显示取餐号、商品摘要、实付金额、预计取餐时间和状态按钮。

订单详情包含订单状态、取餐号、商品快照、联系电话脱敏值、支付记录、退款记录、门店信息和状态时间线。

已取消隐藏二维码；已履约显示核销时间；付费预约取消时显示退款进度。

## 10. 通用异常状态

- 微信支付失败：提供“重新支付”和“查看订单”。
- 支付结果未知：显示“正在确认支付结果”，允许主动刷新。
- 网络失败：保留已填写信息并提供重试。
- 手机号拒绝授权：允许手动输入。
- 购物车跨商户：切换商户时展示该商户独立购物车，不合并商品。
