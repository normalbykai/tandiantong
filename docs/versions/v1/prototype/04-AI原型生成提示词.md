# 摊点通 V1 人工智能原型生成提示词

## 1. 使用方式

先使用“全局提示词”，再追加对应页面提示词。生成结果中的页面标题、菜单、按钮、字段、状态、错误提示、示例商品和人员信息必须全部使用中文。允许保留必要技术标识，但不得生成英文占位文案。

## 2. C 端全局提示词

```text
请为“摊点通”设计微信小程序高保真原型，画板尺寸为 375×812。
产品用于线下零售商品购买和到店服务预约，整体风格温暖、有烟火气、可信赖。
基础颜色使用暖橙、米白和深棕文字，卡片圆角 12 像素，间距清晰，商品图片突出。
底部导航固定为“首页、订单、我的”。
所有界面文案必须使用简体中文，禁止英文占位词和无意义中英文混排。
示例商户使用“春风小铺”，示例商品使用“桂花拿铁”“手作三明治”，示例服务使用“咖啡体验课”。
需要同时设计加载、空数据、失败、禁用和完成状态。
不要加入配送、堂食桌号、套餐、优惠券、会员积分和活动弹窗。
```

## 16. 生成结果检查清单

- 页面是否使用规定画板尺寸。
- 所有界面文案是否为中文。
- 页面编号和名称是否与原型文档一致。
- 是否包含加载、空数据、失败、禁用和完成状态。
- 是否误加入配送、套餐、会员、优惠券、自由装修或平台分账。
- 平台和商户后台菜单是否严格区分。
- C 端订单和预约是否分别进入“订单”和“我的预约”。
