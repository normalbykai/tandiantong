# 摊点通 V1 原型文档索引

## 使用说明

本目录是摊点通 V1 的正式原型规格，用于产品评审、视觉设计、人工智能原型生成和前端开发。全部界面文案必须使用简体中文，流程标注和示例数据也必须使用自然中文。

## 阅读顺序

1. [总体视觉与交互规范](01-总体视觉与交互规范.md)
2. [C 端小程序原型设计](02-C端小程序原型设计.md)
3. [B 端统一后台原型设计](03-B端统一后台原型设计.md)
4. [人工智能原型生成提示词](04-AI原型生成提示词.md)

## 页面编号

| 编号 | 权限域 | 说明 |
| --- | --- | --- |
| `C-xx` | 顾客端 | 微信小程序页面 |
| `B-C-xx` | 后台公共 | 登录、激活、找回密码等公共页面 |
| `B-P-xx` | 平台管理 | 商户开通、审核、平台管理员和审计页面 |
| `B-T-xx` | 商户管理 | 商品、订单、预约、核销、数据和设置页面 |

## 核心流程索引

- 顾客购买：`C-01 → C-04/C-05 → C-06 → C-07 → C-08 → C-09`
- 顾客预约：`C-01 → C-14 → C-15 → C-16 → C-17`
- 商户开通：`B-C-01 → B-P-02 → B-P-03 → B-P-04 → B-P-05`
- 商品上架：`B-T-01 → B-T-02 → B-T-03 → B-T-04`
- 商品核销：`B-T-01 → B-T-10 → B-T-11`
- 预约核销：`B-T-08/B-T-09 → B-T-10 → B-T-11`

## 文档边界

- 本目录定义页面结构、交互与视觉，不替代后端安全校验。
- 隐藏菜单和按钮只改善体验，接口仍必须执行权限和租户校验。
- 本目录不包含配送、套餐、人员排班、自由装修、平台分账和渠道营销。
