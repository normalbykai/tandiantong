# 02 多租户、认证与 RBAC 原型说明

本阶段涉及统一登录、邀请激活、员工与角色管理等页面，但正式页面原型已在 `docs/versions/v1/prototype/03-B端统一后台原型设计.md` 中确认。

本阶段不新增独立页面原型，只约束后端安全行为：

- 统一登录页必须区分“平台管理”和“商户管理”身份页签。
- 平台登录成功只加载平台菜单和平台权限。
- 商户登录成功只加载当前租户菜单、门店上下文和租户权限。
- 员工与角色页面展示的权限树必须来自平台预定义权限，租户管理员不能创建权限标识。
- 前端隐藏菜单和按钮只改善体验，所有受保护 API 仍必须执行后端 RBAC 校验。
