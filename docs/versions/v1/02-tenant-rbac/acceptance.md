# 02 多租户、认证与 RBAC 验收

## 1. 验收范围

- 平台与租户权限域。
- 用户、租户和门店可信上下文。
- 密码哈希与校验。
- 访问令牌签发与解析。
- RBAC 权限判定。
- 安全基础表 Flyway 迁移。
- 租户隔离和权限测试。

## 2. 验收标准

| 项目 | 标准 | 结果 |
| --- | --- | --- |
| 权限域 | 平台用户和租户用户权限域独立 | 通过 |
| 上下文 | 租户和门店来自服务端可信用户归属 | 通过 |
| 密码 | 密码哈希存储，校验不暴露明文 | 通过 |
| 令牌 | 令牌可验证过期时间、用户和权限域 | 通过 |
| RBAC | API 权限由后端判定，两个租户互不影响 | 通过 |
| 迁移 | 安全表具备主键、唯一约束和租户索引 | 通过 |
| 测试 | 阶段相关测试通过 | 通过 |
