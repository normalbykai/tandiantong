# 06 服务预约原型

## 1. B-T-07 服务项目编辑

服务项目采用四步配置：“基础信息、支付与预约规则、排期与容量、预览与上架”。付费服务在支付未验证时只能保存草稿，免费服务可上架。

## 2. B-T-08 预约列表

预约列表展示预约编号、服务名称、预约日期、时段、顾客手机号脱敏值、状态和来源。筛选项包括日期、服务项目和状态。

## 3. B-T-09 预约日历

桌面端显示周历和月历，日期格展示预约数、剩余容量、已满和暂停状态。移动端按日期分组展示日程列表。

## 4. C-15 日期与时段选择

日期横向滚动展示最近 7 天，更多日期使用月历。时段按钮展示时间、剩余名额、价格和不可选原因。已满、已截止和暂停时段保留展示。

## 5. C-17 预约凭证

预约凭证重点展示服务名称、预约日期、时间、门店、联系电话和预约编号。二维码作为辅助核销凭证，已取消时隐藏二维码。
