# 06 服务预约与容量控制验收

## 1. 验收范围

- 服务预约数据库迁移。
- 服务项目、时段容量和预约应用服务。
- 免费预约、付费预约、取消和支付确认。
- B 端和 C 端预约页面原型。

## 2. 验收标准

| 项目 | 标准 | 结果 |
| --- | --- | --- |
| 容量控制 | 最后一个名额后继续预约失败，不出现超额 | 通过 |
| 免费预约 | 确认后立即生成凭证 | 通过 |
| 付费预约 | 创建后待支付，支付成功后生成凭证 | 通过 |
| 取消释放 | 待支付或已预约取消后释放容量 | 通过 |
| 租户隔离 | 非当前租户或门店不能占用或取消预约 | 通过 |
| 构建测试 | 后端测试、前端审计、类型检查和构建通过 | 通过 |
