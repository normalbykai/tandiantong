# V1 外部服务接入说明

## 1. 微信支付

当前代码通过 `WechatPayClient` 定义支付适配接口，并使用 `LocalWechatPayClient` 支持本地预支付、回调验签和退款模拟。真实接入时必须新增生产适配实现，并通过安全配置注入以下信息：

- 微信支付商户号。
- API v3 密钥。
- 商户私钥和证书序列号。
- 微信支付平台证书。
- 支付回调和退款回调公网 HTTPS 地址。

接入后必须补充验签失败、金额不一致、重复回调、超时重试和退款查询测试。禁止仅依据前端跳转结果修改订单状态。

## 2. 微信小程序

C 端首期正式验收平台为微信小程序。真实发布前需要：

- 配置真实 AppID。
- 配置合法请求域名、业务域名和上传下载域名。
- 完成隐私协议、用户授权说明和微信审核资料。
- 在微信开发者工具中执行构建、预览、真机调试和上传。

仓库禁止提交真实 AppID、AppSecret、体验版二维码和生产访问令牌。

## 3. 短信

V1 当前不接入真实短信服务。若启用短信邀请或验证码，必须先确认供应商、签名、模板和成本策略，并新增以下安全能力：

- 验证码有效期和错误次数限制。
- 租户维度限流键。
- 手机号脱敏日志。
- 发送结果审计和失败重试边界。

## 4. 对象存储

V1 当前不接入真实对象存储。接入商户 Logo、头图、商品图和服务图上传时必须：

- 使用临时凭证或后端代理上传。
- 校验文件扩展名、内容类型、大小和图片尺寸。
- 私有读写，按权限生成访问 URL。
- 禁止在日志和响应中暴露存储密钥。
