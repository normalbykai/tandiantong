# 09 联调与验收实施计划

## 1. 功能单元

1. 汇总 01 至 08 阶段提交、验收文档和验证命令。
2. 补齐本地启动说明、Docker 配置说明和外部服务接入说明。
3. 执行后端全量测试、B 端审计、B 端类型检查、B 端构建、C 端审计、C 端类型检查和 Git 空白检查。
4. 更新 09 阶段验收文档，记录真实验证结果、已知限制和待真实凭据验证项目。
5. 提交 09 阶段交付文档，并保留 `codex/v1-development` 分支供用户选择合并、PR 或继续保留。

## 2. 验证重点

- 文档、迁移、测试和前端页面均已纳入版本控制。
- Docker 与本地开发配置不包含真实密钥。
- 真实外部服务未验证项表达清楚。
- 阶段提交保持可独立理解和验证。
