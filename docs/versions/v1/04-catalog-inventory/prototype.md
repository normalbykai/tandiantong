# 04 商品与库存原型

## 1. B-T-02 商品列表与分类

桌面端左侧保持统一后台菜单，主区顶部为分类筛选、状态筛选和“新建商品”按钮。列表展示商品图片、名称、分类、最低价、SKU 数、可售库存、库存预警、上架状态和最近更新时间。

空状态展示“暂无商品”，仅有权限用户显示“新建商品”。

## 2. B-T-03 商品分步编辑

步骤为“基础信息、规格与 SKU、加料配置、库存与上架、预览保存”。

- 基础信息：商品名称、分类、图片、描述和排序。
- 规格与 SKU：规格维度、规格值、组合、价格、SKU 编码和预警值。
- 加料配置：分组名称、必选或可选、最少和最多选择数量、加料项名称和加价。
- 库存与上架：初始库存、是否立即上架、支付配置检查结果。
- 预览保存：展示 C 端商品卡片、规格弹层和价格汇总。

支付未验证时，付费商品的“立即上架”按钮禁用，提示“支付未验证，仅可保存草稿”。

## 3. B-T-04 库存管理与流水

页面包含 SKU 库存表和流水列表。库存调整弹窗要求选择“入库、出库、盘点”，填写数量和中文原因。提交后刷新可售库存、锁定库存和流水。

流水展示时间、类型、商品、SKU、变动数量、业务单号、原因和操作人。

## 4. C-02 商品分类列表

商品优先模板使用顶部横向分类和双列商品卡片；点餐模板使用左侧分类和右侧单列商品。商品卡片显示图片、名称、价格、售罄状态和“选规格”或加号按钮。

## 5. C-04 规格与加料选择

底部弹层展示商品图片、名称、基础价格、规格分组、加料分组、数量和合计金额。必选加料未满足时禁用“加入购物车”；达到最多选择数量时禁用同组其他未选项。
