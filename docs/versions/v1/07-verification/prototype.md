# 07 取餐号与核销原型

## 1. B-T-10 移动核销中心

移动端顶部使用大按钮“扫描顾客二维码”，下方提供“取餐号或手机号后四位”搜索框。待核销列表同时展示商品订单和服务预约卡片。

## 2. B-T-11 核销详情与记录

详情页展示业务类型、商品或服务摘要、时间、顾客脱敏手机号和状态。确认核销前必须二次确认。重复核销展示原核销时间和操作人，不再显示可执行按钮。

## 3. C-08 取餐凭证

取餐凭证核心为大号取餐号，同时展示预计取餐时间、商品摘要、门店地址和核销二维码。页面提示“请勿提前将核销码提供给他人”。

## 4. C-17 预约凭证

预约凭证展示服务名称、预约日期、时间、预约编号和二维码。已取消或已履约时隐藏可用二维码。
