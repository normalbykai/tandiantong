# 07 取餐号与核销验收

## 1. 验收范围

- 核销数据库迁移。
- 取餐号、核销凭证和核销记录服务。
- B 端核销中心与 C 端凭证页面原型。
- 重复核销和租户隔离测试。

## 2. 验收标准

| 项目 | 标准 | 结果 |
| --- | --- | --- |
| 取餐号 | 按租户、门店和营业日生成，仅用于识别 | 通过 |
| 凭证安全 | 核销令牌不可猜测，取餐号不能核销 | 通过 |
| 核销幂等 | 重复核销返回原结果，不重复变更状态 | 通过 |
| 租户隔离 | 非当前租户或门店不能核销凭证 | 通过 |
| 记录留存 | 核销成功和重复核销可查询原记录 | 通过 |
| 构建测试 | 后端测试、前端审计、类型检查和构建通过 | 通过 |
