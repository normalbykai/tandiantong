# 08 经营数据原型

## 1. B-T-12 经营数据

入口位于商户权限域左侧菜单“经营数据”。页面顶部提供日期范围、导出按钮和导出权限状态。默认展示近 7 日数据，移动端保持同一信息顺序并将表格转为纵向卡片。

```text
经营数据
├─ 日期筛选：近 7 日 / 本月 / 自定义
├─ 指标卡：订单数、实收金额、退款金额、待核销
├─ 交易概览：净收入、已支付订单、退款占比
├─ 商品分析：商品销量排行、SKU 排行、加料排行
├─ 预约分析：预约数、取消数、履约数、时段使用率
└─ Excel 导出：当前条件导出、任务状态、文件有效期、审计说明
```

## 2. 状态

- 无数据：显示“当前日期范围暂无经营数据”，导出按钮禁用。
- 无导出权限：展示只读报表，导出按钮禁用并提示“请联系租户管理员开通导出权限”。
- 导出完成：显示文件名、生成时间、脱敏操作人和有效期。
- 导出失败：保留筛选条件并允许重新发起。
