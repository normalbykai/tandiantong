# 03 统一后台与商户开通原型

本阶段遵循 V1 正式原型中的以下页面：

- `B-C-01` 统一登录。
- `B-C-02` 邀请激活。
- `B-P-01` 平台工作台。
- `B-P-02` 商户列表。
- `B-P-03` 新建商户向导。
- `B-P-04` 商户详情。
- `B-P-05` 商户审核工作台。
- `B-P-06` 支付配置。
- `B-P-07` 小程序码管理。
- `B-T-01` 商户工作台。

## 1. 本阶段页面交付

后台首屏不是营销页，登录后直接进入对应权限域工作台。桌面端使用左侧菜单与顶部身份栏；移动端使用抽屉菜单，核销入口优先。

## 2. 页面文案

所有页面标题、菜单、按钮、字段、状态和错误提示使用中文。示例商户使用“春风小铺”，示例联系人使用“张晓春”，示例地址使用“上海路 18 号”。

## 3. 响应式要求

- 桌面端基准宽度 `1440×900`。
- 移动端基准宽度 `390×844`。
- 表格在移动端转为商户卡片。
- 审核、启用、停用和重新邀请管理员必须二次确认。
