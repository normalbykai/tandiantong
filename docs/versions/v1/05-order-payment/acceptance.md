# 05 购物车、订单、微信支付与退款验收

## 1. 验收范围

- 订单支付数据库迁移。
- 微信支付适配接口和本地模拟实现。
- 订单创建、支付回调、取消和退款应用服务。
- B 端和 C 端订单支付页面原型。
- 幂等、状态机、库存和租户隔离测试。

## 2. 验收标准

| 项目 | 标准 | 结果 |
| --- | --- | --- |
| 订单快照 | 保存商品、SKU、加料名称和成交价格快照 | 通过 |
| 库存锁定 | 创建订单锁定库存，取消释放，支付扣减，退款回补 | 通过 |
| 支付安全 | 回调必须验签，失败不改变订单状态 | 通过 |
| 幂等 | 重复创建、重复回调和重复退款返回一致结果 | 通过 |
| 退款范围 | 仅支持核销前整单退款，已完成订单禁止在线退款 | 通过 |
| 构建测试 | 后端测试、前端审计、类型检查和构建通过 | 通过 |
