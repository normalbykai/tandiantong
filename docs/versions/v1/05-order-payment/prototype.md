# 05 订单与支付原型

## 1. C-05 购物车

购物车按当前商户独立保存。商品明细展示商品名、SKU、加料、单价、数量和小计。商品下架、售罄或价格变化时展示“已失效”，顾客必须删除或重新选择。

## 2. C-06 确认订单

确认订单展示联系电话、预计取餐时间、商品快照、商品金额、加料金额、实付金额、门店地址和退款说明。底部按钮为“提交订单并支付”，提交中禁用重复点击。

## 3. C-07 支付结果

支付成功展示“支付成功 · 待取餐”、取餐号、商品摘要、门店地址和“查看订单详情”。支付失败展示“重新支付”和“查看订单”。支付结果未知展示“正在确认支付结果”。

## 4. B-T-05 订单列表

桌面端使用状态页签、订单号搜索、手机号后四位筛选、时间筛选和表格。移动端转为卡片，突出取餐号、金额、预计取餐时间和状态。

## 5. B-T-06 订单详情

顶部展示订单状态、取餐号、金额、创建时间和操作。下方页签为“订单信息、支付退款、核销记录、操作日志”。手机号默认脱敏；整单退款必须填写原因并二次确认。
