# 第三阶段预约完整闭环验收记录

## 1. 验收范围

本记录对应 V1.1 第三阶段“预约完整闭环”的后端基础能力验收，覆盖预约支付确认、超时取消、核销履约、顾客查询与取消，以及关键幂等和状态机边界。

## 2. 已覆盖自动化测试

| 范围 | 覆盖情况 |
| --- | --- |
| 预约支付确认 | 覆盖付费预约回调确认、错误金额拒绝、已确认重复回调不重复签发凭证、已取消预约回调拒绝 |
| 预约超时取消 | 覆盖过期待支付预约取消后释放容量、并发下已被取消时不重复释放容量 |
| 预约核销履约 | 覆盖预约凭证分发到预约完成处理器、`CONFIRMED -> FULFILLED` 条件更新、非确认状态拒绝履约 |
| 重复核销 | 覆盖已核销预约凭证重复提交不重复写核销记录、不重复推进预约状态 |
| 顾客查询与取消 | 覆盖按租户门店和联系电话查询列表、手机号不匹配详情不可见、手机号不匹配取消不释放容量、本人取消释放容量 |
| 凭证非法状态 | 覆盖已取消凭证不能核销且不触发业务完成处理器 |
