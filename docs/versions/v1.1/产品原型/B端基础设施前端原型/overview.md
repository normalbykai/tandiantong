# 摊点通 · 第四版「墨璃」设计升级总览

## 升级背景

基于第三版「青墨」设计系统，在保持原有功能、结构与双权限域不变的基础上，全面升级视觉冲击力、辨识度与交互细节。

**版本号**：v1.1.0 → **v1.2.0**
**设计系统**：青墨 → **墨璃（Ink-Glass）**
**目录**：`第四版/`

---

## 五大设计升级

### 1. 玻璃拟态（Glassmorphism）
- 卡片、抽屉、弹窗、顶栏全部启用 `backdrop-filter: blur + saturate`
- 半透明背景 + 渐变边框 + 内嵌高光，呈现"墨色玻璃"质感
- 登录卡、Toast、Modal、Drawer 均应用

### 2. 动态光效（Aurora / Glow）
- **登录页**：极光渐变背景（auroraMove 动画 18s 循环），辅以两个发光球斑
- **侧边栏**：顶部径向光晕，按权限域染色（墨绿/墨青）
- **英雄数据区**：深色渐变 + 网格纹理 + 径向辉光
- **统计卡**：hover 时右侧浮现径向光斑
- **菜单选中项**：左色条带发光投影
- **Logo 标识**：渐变 + 内嵌高光 + 光晕投影

### 3. 微交互动效（Micro-interactions）
- **按钮涟漪**：所有 `.btn` 点击时从点击点扩散涟漪
- **菜单项滑入**：侧边栏菜单项依序 80ms 间隔入场
- **统计数字滚动**：进入视口时数字从 0 平滑增长到目标值
- **头像在线指示器**：呼吸光晕动画
- **通知红点**：脉冲扩散动画
- **关闭按钮**：hover 时旋转 90°
- **图标 hover**：scale + rotate 微动
- **顶栏图标**：hover 时 translateY
- **权限点类型 dot**：发光阴影
- **时间线节点**：脉冲呼吸
- **键盘快捷键 Ctrl+K**：触发全局搜索 Toast

### 4. 数据可视化（Data Viz）
- **平台工作台**：新增"近 7 天操作趋势"条形图（mini-bars 渐变填充，hover 上浮）
- **英雄数据区（hero-stats）**：4 列数据卡，玻璃质感 + 网格背景 + 径向辉光
- 数字使用 `font-mono` + 渐变文字
- 表格行 hover 渐变背景（从左到右淡入）

### 5. 现代圆角与阴影系统
- 圆角整体放大：5px → 7px（按钮）/ 8px → 12px（卡片）/ 14px → 16px（弹窗）/ 18px（抽屉）
- 阴影改为多层通透：`shadow-md/lg/xl` 均为 2 层叠加，带主色相
- 按钮 hover 上浮 + 主色光晕
- 选中项/标签使用渐变填充而非纯色

## 技术实现要点

### 新增 CSS 类
- `.hero-stats` / `.hero-item`：英雄数据区
- `.quick-grid` / `.quick-card`：快捷入口卡片
- `.stat-spark`：迷你趋势图
- `.btn-ripple`：按钮涟漪效果
- 渐变变量：`--plat-grad` / `--merc-grad` / `--accent-grad`

### 新增 JS 函数（`shared.js`）
- `attachRipple()`：按钮涟漪（含 MutationObserver 兼容动态插入）
- `animateMenuItems()`：菜单项依次入场
- `animateStatValue()` + `initStatAnimations()`：数字滚动（IntersectionObserver）
- `initTableRowSelection()`：表格行点击高亮
- `initKeyboardHints()`：Ctrl+K 快捷搜索
- `initLiveClock()`：实时时间更新

### 保持兼容
- 所有原有 class 名称保留，HTML 结构无需改动即可受益于升级
- 双权限域 `[data-domain="platform|merchant"]` 机制保留
- `shared.js` 中的 DRAWERS / MODALS 模板完全兼容
- 仅需修改版本号与 ver-tag 文案

## 设计哲学

> 「青墨」是克制的企业感，「墨璃」是在克制中点亮呼吸——
> 同样的墨色基底，但有了玻璃的通透、光线的流动、数字的脉动。
> 不是颠覆，是让原本安静的设计，活起来。
