商户角色
管理本商户内的角色与权限配置。商户管理员只能配置本商户范围内的权限，不能授予平台权限。商户角色仅适用于本商户员工。
商户角色与平台角色属于不同权限域。商户角色只能配置本商户范围内的权限点（工作台、门店信息、员工账号、商户角色、商户权限、操作日志、个人设置），不能授予平台管理端的任何权限。
