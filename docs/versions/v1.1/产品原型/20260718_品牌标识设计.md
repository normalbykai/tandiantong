# 摊点通 V1.1 品牌标识设计

## 设计目标

摊点通服务线下零售与到店服务商户，品牌标识应同时表达“摊点”的亲和力和交易、预约、核销等流程的可靠性。标识需要适配顾客微信小程序、商户响应式后台与平台管理端，并在小尺寸下保持可识别。

## 项目分析结论

- 目标使用者同时包含顾客与商户经营者，视觉不能只偏向餐饮，也不能过度企业化。
- 当前业务以商品购买、预约和核销为核心，品牌应传递流程顺畅、结果可信和线下服务可达。
- 既有标识以橙色摊棚和绿色线条为主，色彩方向贴合产品；其中听诊器式轮廓容易被理解为医疗服务，不适合作为覆盖零售与到店服务的主图形。

## V4 主标识方案

图形不再直白描绘摊棚或交易勾选，而是以“点穿过通道”为核心隐喻。一个橙色圆点停在青绿色开放方框的入口，表达“摊点”与“打通”的双重含义；开放角同时留下可呼吸的留白。这一结构更接近数字产品标识，在小尺寸下也容易识别。

图形由三个部分组成：

1. 橙色圆点表示摊点、交易触点与被看见的商户。
2. 青绿色开放方框表示系统将顾客、商品、订单、预约和核销串成一条可通行的路径。
3. 入口留白让图形更有呼吸感，也让“通”成为可见的视觉动作。

主图形不包含文字，以避免小程序图标、浏览器图标和窄导航中出现文字不可读的问题。横向组合版本应在图形右侧配用“摊点通”中文标准字；标准字建议采用字面清晰的现代无衬线中文字体，并保持深墨色。

## 色彩与使用规则

| 用途 | 色值 | 含义 |
| --- | --- | --- |
| 触点主色 | `#FF6B4A` | 温度、行动感与线下商户活力 |
| 通道主色 | `#0B625C` | 可信、秩序与数字化连接 |
| 标准字与深色背景 | `#163D3A` | 稳重且保证可读性 |
| 浅色页面底色 | `#FFFFFF` | 保持后台与小程序界面清爽 |

- 图标四周至少保留图标宽度的 12% 留白。
- 小于 32 px 的场景可省略台面内两侧圆点，仅保留勾选和棚檐轮廓；本阶段先使用完整图形。
- 禁止拉伸、旋转、添加阴影或渐变；禁止将颜色替换为与状态色相混淆的红色、黄色或纯绿色。
- 深色背景上使用原色图标；浅色背景上优先使用原色图标。单色印刷时使用 `#1F2937`。
