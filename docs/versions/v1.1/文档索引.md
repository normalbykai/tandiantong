# 摊点通 V1.1 文档索引

## 版本定位

V1.1 是当前开发版本，以 Java 21、Spring Boot 3、MyBatis-Plus、Sa-Token、Knife4j 和 Maven 多模块的模块化单体为技术基线。该版本继承 V1 已确认的业务边界，重点继续完成交易闭环、权限细化、外部服务接入和自动化验收，不提前引入微服务或未确认业务。

## 当前状态

| 范围 | 状态 | 说明 |
| --- | --- | --- |
| 持久层重构 | 已完成 | 业务服务不再使用 `JdbcTemplate`，统一使用 MyBatis-Plus |
| 认证重构 | 已完成 | 自研 JWT 已替换为 Sa-Token |
| 接口文档 | 已完成 | 后台与小程序控制器已接入 Knife4j |
| 交易闭环补强 | 待开发 | 包含超时释放、付费预约、退款重试和完整审计 |
| 权限细化 | 待开发 | 从数据库加载权限码并执行接口级授权 |
| 外部服务 | 待开发 | 微信登录、真实微信支付、对象存储和短信按确认范围接入 |
| 全链路验收 | 待开发 | 双租户、并发、幂等、异常重试和部署恢复验证 |

## 文档维护规则

- 新文档必须放入 V1.1 对应中文目录，文件名和正文必须使用简体中文。
- 技术标识、类名、接口路径、数据库字段和标准协议可以保留英文。
- 每个功能单元必须同步更新设计、实现状态和验收证据。
- V1 文档只读，不得继续作为当前实现依据。
