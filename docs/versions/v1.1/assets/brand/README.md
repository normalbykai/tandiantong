# 摊点通品牌资产 V4

本目录的 V4 文件为当前可直接引用的 SVG 源文件。V3 和此前位图均为已淘汰的探索稿，不应用于新页面。

| 文件 | 适用场景 |
| --- | --- |
| `tandiantong-logo-mark-v4.svg` | 小程序图标、浏览器图标、折叠侧栏、加载页 |
| `tandiantong-logo-horizontal-v4.svg` | 后台登录页、网页顶部导航、正式文档页眉 |
| `tandiantong-logo-vertical-v4.svg` | 海报、启动页、空间充足的登录页 |
| `tandiantong-logo-horizontal-reverse-v4.svg` | 深色或照片背景；不得用于浅色背景 |

## 前端引用

```html
<img src="/assets/tandiantong-logo-horizontal-v4.svg" alt="摊点通" />
```

图形标识最小显示尺寸为 24 px；横版标识最小高度为 28 px。图形四周保留至少图形宽度 12% 的留白，横版标识不得拆开图形与文字、不得拉伸、描边、加阴影或改色。
