# 平台系统管理设计

## 范围

本功能属于平台权限域。平台管理端新增系统管理入口，第一阶段提供平台基础设置和平台通用字典；商户管理端仅预留独立入口，不读取平台配置和平台字典。

## 平台基础设置

- Logo 使用图片地址保存，暂不实现文件上传和对象存储。
- 平台描述限制为 255 个字符。
- 配置使用固定主键保存为全局平台数据，不包含 `tenant_id`。
- 修改配置需要平台系统配置权限，并写入操作审计记录。

## 平台字典

- 字典项包含字典类型编码、字典项编码、字典项颜色类型、字典项名称、排序值和启用状态。
- 字典项编码用于程序识别，颜色类型用于页面展示；字典项名称用于页面展示，不再保留重复的业务实际存储值字段。
- `(dictionary_type, item_code)` 必须唯一。
- 字典项不物理删除，停用后保留历史数据。
- 查询和写入接口仅位于 `/api/platform/v1/system/**`，通过平台权限码鉴权。

## 后续范围

对象存储接入后，可将 Logo 图片地址配置替换为受控上传流程；商户系统管理功能需另行设计商户级配置和租户隔离规则。
