# 平台权限与商户生命周期开发总结

## 本次范围

本次完成平台权限管理、商户管理员首次激活和租户生命周期管理相关能力。

## 平台权限管理

- 新增平台账号、平台角色和平台权限点管理页面与接口。
- 支持平台账号新增、编辑、启停和密码重置。
- 支持平台角色新增、编辑、启停和权限配置；受保护管理角色可避免因普通权限调整而失去恢复管理能力。
- 新增权限数据迁移和操作审计，平台角色权限仍由后端接口作为最终安全边界。

## 商户开通与激活

- 新增商户管理员首次激活页面 `/merchant/activate`，通过邀请码设置首次登录密码。
- 商户管理端登录页增加激活入口；激活完成后自动回到已选中的商户管理端登录页。
- 登录页默认填充 `admin` / `admin`，并支持“7 天内自动登录”。
- 前端统一请求层兼容 `204 No Content`，避免启用商户等无响应体接口触发 JSON 解析错误。

## 租户生命周期管理

- 租户管理支持启用和停用；停用后商户后台登录及后续请求均会受租户状态校验拦截。
- 待激活商户可以重新生成管理员邀请码；旧的未使用邀请码会立即失效，新的明文邀请码仅在接口响应中展示一次。
- 已激活商户禁止重发邀请码，防止重复创建管理员账号。
- 停用商户和重新生成邀请码会记录操作审计。

## 数据库迁移

- `V2_20260718_7`：商户停用与邀请码重置权限。
